<h1 align="center">Sixty Barbershop</h1>

# About
This web application is a combination of a company profile integrated with dynamic booking system and payment gateway.
This website displays information such as the services offered, the barbers who work at the shop, as well as the barbershop’s location and operating hours.

The main problem of application is that services have various prices depending on each barber’s expertise and experience. 
To solve this, the system implements a dynamic pricing mechanism that allows different prices for the same service based on the selected barber.
In addition, the booking schedule is automatically adjusted according to the duration of each service to prevent overlapping appointments. 
Customers can make bookings by selecting a service and a barber, choosing an available date and time slot, and completing the payment through Midtrans integration.

On the admin panel, administrators can monitor booking schedules and manage service and barber data, including setting dynamic prices 
for each service and barber combination. Administrators can also disable services that are temporarily unavailable or deactivate
barbers who are on leave or no longer working.
